# Expense Tracker Telegram Bot

Telegram-бот для учёта расходов с голосовым и текстовым вводом.
Извлекает сумму, категорию, способ оплаты и банк через OpenAI,
сохраняет данные в Google Sheets.

## Возможности
- Текстовые и голосовые сообщения
- Локальное распознавание голоса (Faster-Whisper)
- Автоопределение способа оплаты и банка
- Уточняющий диалог, если банк не указан
- Команда `/report` – аналитика за месяц
- Многопользовательская поддержка (один лист Google Sheets)

## Установка
1. Клонируйте репозиторий
2. `pip install -r requirements.txt`
3. Скопируйте `.env.example` → `.env` и заполните ключи
4. Поместите `credentials.json` в корень
5. `python bot.py`

## Использование
Отправьте боту «Кофе 250 рублей карта Тинькофф» или голосовое сообщение.
Данные автоматически попадут в таблицу.
Команда `/report` покажет сводку за текущий месяц.
